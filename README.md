An Online Food Ordering System built using PHP, MySQL, HTML, CSS, and JavaScript. This project allows users to browse food items, add them to cart, and place orders online, while admins can manage menu items and orders efficiently.

👤 User Side
User Registration & Login
Browse Food Items
Add to Cart / Remove from Cart
Place Orders
View Order History

🛠️ Admin Side
Admin Login
Add / Update / Delete Food Items
Manage Orders
View Users

🧑‍💻 Technologies Used
Frontend: HTML, CSS, JavaScript
Backend: PHP
Database: MySQL
Server: XAMPP / Apache
